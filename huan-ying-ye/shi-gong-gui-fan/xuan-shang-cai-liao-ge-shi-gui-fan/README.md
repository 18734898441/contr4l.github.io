# 悬赏材料格式规范

1. 必须按照Markdown语法提交
2. 提交文件必须以`.md`作为后缀名，语法规则请参考子链接中的Markdown基本语法
   1. 必须要遵守的语法格式包括N级标题、加粗、下划线
   2. 如果内部有超链接内容，请在开头注明TODO项，将由前端统一处理
3. 内容模板

```
提交人：Nickname[Torn id]
提交日期：20xx-xx-xx
原文地址：www.tornwiki.com/example
内部超链接：
- [ ] 未完成链接一
- [ ] 未完成链接二
- [ ] 未完成链接三
```
